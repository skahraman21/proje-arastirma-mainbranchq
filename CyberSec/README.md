## Hedefler
- Başarılı ve açık kaynaklı güvenlik araçlarını/projelerini **Fork** ederek kodlarını incelemek, mantığını anlamak ve kendi senaryolarıma göre modifiye etmek.
- Güvenli kodlama mantığını kavrayabilmek adına başkalarının projelerindeki açıkları veya savunma mekanizmalarını test etmek.
